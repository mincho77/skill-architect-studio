## Notas

- **Variantes:** sufijos `-alt`, `-s`, `-linea`, `-tono` son versiones del
  mismo ícono (alternativa, simplificada/small, solo línea, con tono).
  Para **16 px** prefiere la variante simplificada (`-s` / `-linea`) por
  legibilidad (regla 'Iconos simplificados' del manual).
- **Estados batería/señal:** `bateria-0/10/20/60/80/100`, `barras-senal`,
  `ondas-senal` representan niveles.
- **`prohibido-*`:** versión tachada de la acción correspondiente.
- Errores de tipografía heredados del banco se conservan **literales**:
  `calendariio-cruz`, `interrogracion`, `reiodactivo-alt`, `flecha-derec-linea`,
  `avant-derech`, `avant-izquier`. Referencia el nombre tal cual.

## Reglas operativas para el skill

- **Cómo elegir un ícono:** `grep` por palabra clave sobre esta lista (la
  agrupación por categoría es heurística, no oficial) → toma el nombre exacto →
  usa `assets/icons/{nombre}.svg`. No inventes nombres: si no está en la lista,
  el ícono no existe en el set.
- **Variantes por sufijo:** `-alt` (alternativa), `-s` (simplificada/small),
  `-linea` (solo línea), `-tono` (con tono). Para **16 px** prefiere `-s` o
  `-linea` por legibilidad; para tamaños grandes, la versión completa.
- **Nombres literales:** algunos traen erratas heredadas del banco
  (`calendariio-cruz`, `interrogracion`, `reiodactivo-alt`, etc.). Cópialos
  tal cual — corregir la ortografía rompe la referencia al archivo.
- `prohibido-*` = versión tachada de la acción; `bateria-0…100` /
  `barras-senal` / `ondas-senal` = niveles. Para reglas de **uso/color/tamaño**
  de íconos, ver `iconos.md` (este archivo es solo el índice de nombres).
